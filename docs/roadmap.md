# Roadmap

## MVP: deployable personal engineering website

Target outcome: a polished, content-first website that can be shared with hiring managers and technical leaders.

### Must have

- Home page
- Projects index
- Field Notes index
- About page
- Resume page or resume link
- Responsive layout
- Basic SEO metadata
- Clean navigation and footer
- Cloudflare Pages deployment

### Initial project case studies

- Building a Multi-Tenant Security Analytics Platform
- BlindSpot Labs
- Executive Reporting Automation
- AI Security & Agent Workflows

### Initial field notes

- Why AI Agents Need Identity
- MCP Is Not an Identity Solution
- API Security Lessons Applied to AI Systems

## Phase 2: content depth

- Add diagrams to key project pages
- Add tags and topic pages
- Add RSS feed
- Improve Open Graph/social previews
- Add downloadable resume
- Add CotaLabs link and project relationship model

## Phase 3: polish and scale

- Add search if content volume justifies it
- Add talks/presentations section
- Add selected demos or labs
- Add link checking and build validation in CI
- Add accessibility checks

## Explicitly deferred

- CMS
- Database
- Authentication
- Comments
- Newsletter
- Heavy animation
- Complex analytics
