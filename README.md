# alancota.net

Personal website, engineering notebook, and technical portfolio for Alan Cota.

This site documents engineering projects, architecture decisions, technical writing, and experiments around Identity, API Security, AI Systems, Customer Engineering, and Platform Engineering.

## Purpose

This is not intended to be a generic online resume. The goal is to document how I think through technical problems, build reusable patterns, and help enterprises adopt emerging technology safely.

## Planned stack

- Astro
- TypeScript
- Tailwind CSS
- MDX
- Cloudflare Pages

## Content model

- **Projects** — anonymized case studies and technical project writeups
- **Field Notes** — short-form technical writing and observations
- **About** — professional background and operating principles
- **Resume** — downloadable resume and professional links
